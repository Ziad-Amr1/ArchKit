# ArchKit

ArchKit is a modern frontend web application built with React, focused on providing a clean and minimal digital marketplace experience.

The project is structured with scalability in mind and follows a component-based architecture.  
It currently includes the core layout, navigation bar, and a hero section with a carousel, serving as the foundation for future features.

## Tech Stack
- React
- Vite
- Tailwind CSS
- React Slick

## Project Status
The project is under active development and will be expanded incrementally.
